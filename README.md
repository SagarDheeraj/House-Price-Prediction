# 🏠 House Price Prediction

## 📌 Overview
A machine learning project to predict house prices 
using the California Housing dataset.

## 🛠️ Technologies Used
- Python
- NumPy, Pandas
- Scikit-learn
- Matplotlib, Seaborn

## 🤖 Algorithm
- Linear Regression
- Random Forest Regressor

## 📊 Model Performance
- Linear Regression Accuracy: 57%
- Random Forest Accuracy: 80.51%

## 📂 Dataset
- Name: California Housing Dataset
- Source: Built-in Sklearn Dataset
- Rows: 20,640
- Features: 8

## 📈 Visualizations
- Feature Importance
- Price Distribution
- Actual vs Predicted
- Correlation Heatmap

## 🔍 Key Findings
- Median Income is the most important 
  feature for predicting house prices
- Random Forest performed significantly 
  better than Linear Regression
- No missing values found in dataset

## 👨‍💻 How to Run
1. Clone this repository
2. Open House_Price_Prediction.ipynb
3. Run all cells in order
